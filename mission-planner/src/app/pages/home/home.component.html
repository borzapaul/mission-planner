Steps to Serve & Build the App:

1. Clone the Repository:

git clone https://github.com/borzapaul/mission-planner.git

2. Navigate to the Project Directory:

cd mission-planner

3. Install Dependencies:

npm install

4. Serve the App (Development Mode):

This will start a development server. Open your browser and navigate to http://localhost:4200/ to view the app. 

ng serve

5. Build the App (Production Mode):

This command will compile the app and generate a dist folder with the production-ready files. 

ng build --configuration production

Steps to Use the App:

Navigation:

  • Click on the Mission Planner logo to navigate to the home page.
  • Use the hamburger menu on the top-left to navigate between different pages (Home, Planner, Player).

Home Page:

The home page provides an overview of the app and contains step-by-step instructions for serving and building the app.

Planner Page:

The planner page allows users to create and manage mission plans.

Player Page:

The player page enables users to execute and simulate mission plans.

Additional Notes:

  • Ensure you have Node.js and Angular CLI installed on your system before following the steps.
  • For detailed Angular CLI documentation, refer to: Angular CLI Documentation.
